# 视频发布前端页面

这是一个用于多平台视频发布的 React 前端应用。

## 功能特性

### 左侧面板 - 发布信息
- **标题输入**：输入视频标题
- **副标题输入**：输入视频副标题
- **视频上传**：拖拽视频文件到指定区域，获取文件路径
- **标题图片上传**：拖拽图片文件到指定区域，获取文件路径

### 右侧面板 - 发布设置
- **平台选择按钮**：
  - 抖音
  - 小红书
  - B站
  - 西瓜视频
  
- **单平台发布按钮**：分别发布到各个平台

- **主要操作**：
  - **一键填充 (Preview)**：将数据填充到各平台预览
  - **一键发布**：同时发布到所有选中的平台

## 安装和运行

```bash
# 安装依赖
npm install

# 启动开发服务器
npm run dev

# 构建生产版本
npm run build
```

## 技术栈

- React 18
- Vite
- CSS3

## 开发说明

项目使用 Vite 作为构建工具，支持热更新。修改代码后会自动刷新页面。

## 后续集成

此前端页面可以与后端 Python 脚本集成，通过 API 调用实现真实的视频发布功能。
